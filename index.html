My Projects.

Line Encoder and Scrambler Using Python |GitHub

Line Coding Schemes was a part my digital communication course at college and I thought implementing these might result in better understanding of the underlying concepts. Line coding is the process of converting digital data to digital signals. It refers to the waveform pattern of voltage or current used to represent the 1s and 0s of a digital signal on a transmission link. In this project I have implemented line coding schemes including NRZ-L, NRZ-I, Manchester, Differential Manchester, AMI and scrambling techniques like HDB3 and B8ZS using Python.

Spam Filter |GitHub

Developed a model on Natural Language Processing (NLP) for the filtration of spam emails in order to enhance online security. It used Multinomial Naive Bayes Algorithm to block spam emails based on the sender as well as the content of the mail. It had an accuracy of 99%.
